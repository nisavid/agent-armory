# Keep Seed Examples Demonstrative

Forge Seed examples will be annotated demonstrations of the decision method, with realistic file shapes and explicit non-production status. They can serve as starting points, but they are not installable Agent Equipment until a Smith promotes them through the full capability workflow.

## Considered Options

- Make examples annotated demonstrations of the Forge method.
- Make examples complete copyable equipment packages.
- Omit examples from the Forge Seed.

## Consequences

- Examples can teach the method without implying untested production readiness.
- Each example should trace from capability card to interface decision record to projected components.
- The promotion path from Forge Example to Agent Equipment must remain explicit.
